Configuration Generator

Generate optimized YAML configurations for your Instana agents based on your service's technology stack and optimization goals.

Note: The fields and values in filters correspond to span annotations and their values. For a complete reference of available JDBC span annotations, see the JDBC Attribute Mapping page.

1. Select Technology

Select the technology stack used by your service

2. Define Filter Mode

Choose whether to exclude or include spans that match your filter criteria

3. Choose Filter Preset (Optional)

Select a preset configuration or create custom filters below

4. Define Custom Filters

Create custom attribute filters to fine-tune your optimization

          - key: "db.system"
            value: "jdbc"
